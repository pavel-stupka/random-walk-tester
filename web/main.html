RWTester
software for testing a random walk

Introduction

RWTester is a piece of software for testing a random walk algorithm and evaluating obtained data. Main goal of this project is to examine behaviour of the random walk on different kinds of graphs. RWTester is a part of the project aimed at graph traversal by random walk problem.

RWTester is distributed under the terms of the GNU General Public Licence version 2

News

RWTester features

  • Generating graphs (scale-free graphs, random graphs, complete graphs, trees)
  • Analyzing graphs
  • Running a random walk (cover and find path modes)
