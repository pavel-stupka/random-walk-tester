RWTester
software for testing a random walk

Changes

  • 0.1alpha —> 0.1

    RWTester can now generate random graphs using a -generate command. For instance this command:

    rwtester -generate R100-200

    will generate a random graph called R100-200.graph with 100 vertices and 200 edges that are chosen randomly.

    Also two new data files are produced as a result of the random walk analysis. These are template_length_time.txt and template_length_visited.txt.
