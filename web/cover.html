RWTester
software for testing a random walk

Random walk — cover mode

Coverage of a graph means how many vertices were allready visited (at least once) by the random walk. For example 50% of a graph coverage means that exactly one half of all vertices were at least once visited by the random walk. We can run a random walk and let in process the graph until it reaches some given graph coverage. This is exactly what the cover mode does.

We want to test a random walk on this graph (sf.graph). This graph is a scale-free network generated by RWTester using this command: 

rwtester -generate SF5-1000 -template sf

To start testing the random walk we run this command: 

rwtester -mode cover -input sf.graph -loop 100 -coverage 100 -start 0 -template sf
  • -mode cover — We run the random walk in a graph cover mode
  • -input adding.graph — Our tested graph
  • -loop 100 — The results should be generated as an average of 100 tests
  • -coverage 90 — The random walk should achieve 100% of a graph coverage
  • -start 1 — The random walk starts from the vertex named '0'
  • -template adding — A template for saving results to the files
