software for testing a random walk

Graph file format

This tutorial describes a graph file format that RWTester can work with. Graphs are stored in plain text files having a suffix .graph. Each line of such a file can be either a vertex, an edge or a comment.

Here are some examples

  • Vertex[A]
  • Edge[A] -> [B]
  • Comment# This is a comment

Example of a graph file

# directed edge from A to B with a weight = 5
[A] -> [B] 5

# directed edge from B to A with a weight = 3 
[A] <- [B] 3

# undirected edge between A and B with a weight = 1
[A] -- [B] 1

# undirected edge between A and B with a default weight = 1
[A] -- [B]

# vertex C
[C]

Strict notation

Each line of a graph file can be described by the regular expression from the theory of formal languages.

  • A = printable characters
  • C = printable characters except '[' and ']'
  • W = white space characters
  • I = digits

Edge:
W*[C+]W*("--" + "->" + "<-")W*[C+](W* + W*I+W*) Vertex:
W*[C+]W* Comment:
W*#A* Empty line:
W*
